📚 How to Use

Try clicking on any word in this document, or select text and a context menu will appear. For this demo, click on any word to see dictionary definitions in the popup.

💡 Interaction: Click on a word → Lookup popup appears → Browse definitions from different dictionaries

Sample Words to Try

Click on any of these words to see the dictionary popup: book, dictionary, knowledge, science, language

The system supports Urdu dictionaries as well. Try words like: علم (knowledge), کتاب (book), لغت (dictionary)

🔍 Features

✓ Ultra-fast lookups - Binary search in <1ms
✓ Multiple dictionaries - View tabs for each dictionary
✓ Lazy loading - Only loads indices on demand
✓ Memory efficient - ~25MB max memory usage
✓ Mobile friendly - Works on low-end Android devices
✓ Black & white UI - Clean, minimal design
