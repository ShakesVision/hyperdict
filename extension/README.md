# HyperDict — Chrome Extension (MV3)

Turns **any webpage** into a dictionary: select a word (or use the toolbar
button) and get instant StarDict definitions in the reusable HyperDict popup.
It's the same engine + UI as the library, driven by a settings page.

Lives in this repo (no separate repo) and reuses the built bundles.

## How it works

- `content.js` runs in every page's isolated world (after the bundled
  `vendor/hyperdict.min.js` + `vendor/hyperdict-ui.min.js` + `settings.js`),
  reads your settings from `chrome.storage.sync`, and — if enabled for that host
  — builds the engine and mounts the popup + selection chip.
- `options.html` is the full settings page (appearance + dictionary editor).
- `popup.html` is the toolbar popup: open the dictionary, toggle the current
  site, or jump to settings.
- `settings.js` is shared, dependency-free logic (defaults, config building),
  also covered by the repo's tests so we don't ship a broken extension.

## Load it locally (development)

```bash
npm install
npm run build          # compiles bundles into extension/vendor/ (and docs/)
```

Then in Chrome:

1. Open `chrome://extensions`
2. Enable **Developer mode** (top-right)
3. **Load unpacked** → select this `extension/` folder
4. Pin the HyperDict icon; open any page, select a word → tap the 🔍 chip, or
   click the toolbar icon → **Open dictionary**.

> Re-run `npm run build` after changing library code, then hit **Reload** on the
> extension card. Changing only extension files (`content.js`, `options.js`, …)
> just needs the **Reload**.

## Settings

Toolbar popup: open the dictionary, enable/disable on the current site.
Options page (right-click icon → Options, or the popup's "Settings…"): direction,
placeholder, selection-chip label, attribution, long-press, and the full
**dictionary list** (add/remove; each is a folder URL or an archive URL, with
language/direction/font). Everything syncs via `chrome.storage.sync`.

## Icons

`icons/icon{16,48,128}.png` are generated by `icons/generate.mjs`
(`node extension/icons/generate.mjs`) — a plain magnifier. Replace with branded
artwork anytime (keep the same filenames/sizes).
